# Project 3: Around The U.S.
  
**Intro** 
This project is to make a one page website by using some of the pictures from the figma. Website should be look like the same as the figma one shows. All the files is organized according to FLat BEM principles. Some of the pictures are resized by https://tinypng.com/  The website's layout can adapt to different screen sizes and devices from 320px to over 1280px. To help creating such responsive website, is to use either grid layout or flexbox.
